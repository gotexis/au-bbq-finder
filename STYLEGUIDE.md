# Style Guide

## 设计参考
- [https://www.willyweather.com.au](https://www.willyweather.com.au) — 借鉴了地理数据的分州浏览结构、地图交互模式
- [https://toilet.rollersoft.com.au](https://toilet.rollersoft.com.au) — 借鉴了 Leaflet 聚合地图 + 过滤器的交互模式（同系列站点）

## 配色方案
DaisyUI theme: autumn
Primary accent: amber-700 to orange-900 gradient (BBQ/fire vibes)
Stats: amber-700 (BBQ), green-700 (picnic), blue-700 (total)

## 设计决策
- 暖色调（琥珀/橙）呼应BBQ/户外主题
- Marker clustering 处理 23k+ 数据点
- 分州浏览 + 全国地图两种入口
- 过滤器：类型（BBQ/picnic）+ 燃料（electric/wood/gas）
- 移动优先，地图全屏宽度
